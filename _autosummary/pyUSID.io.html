pyUSID.io¶

Tools to read, write data in h5USID files

Submodules¶

hdf_utils

Utilities for reading and writing USID data in HDF5 files

dtype_utils

Utilities for transforming and validating data types

image

ImageTranslator class that translates conventional 2D images to USID HDF5 files

io_utils

Utilities for formatting strings and other input / output methods

numpy_translator

ArrayTranslator capable of translating numeric arrays to USID HDF5 files

usi_data

USIDataset class that simplifies slicing, visualization, reshaping, etc.

write_utils

Utilities that assist in writing USID related data to HDF5 files
