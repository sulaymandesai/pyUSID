Examples & Tutorials¶

  • The documents below should be sufficient for you to learn how to use and even write code for pyUSID.

  • If you haven’t already, please go through the tutorials on the basics of python, numerical analysis, tools like jupyter notebooks, etc.

  • All the documents listed below will assume familiarity with the Universal Spectroscopy and Imaging Data (USID) model. Please read that document before beginning

  • Please get in touch with us if there are topics that you think need to be clarified / expanded / deserve new tutorials, or if you find any errors.

  • Please refer to our sister package - pycroscopy for examples on scientific data analysis and papers generated using pyUSID and pycroscopy.
